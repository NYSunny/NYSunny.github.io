Github作为图床

尝试把图片上传Github，以Github作为图片服务器，在本地markdown文件中引用图片链接，上传到Github上的图片链接统一为如下格式：

https://raw.githubusercontent.com：固定地址

NYSunny：Github账户名

imgs-1：存放照片的仓库名

master：存放照片的仓库主分支

Java.jpg：照片名

markdown引用Github上的图片十分不稳定，有时能够显示，有时不能显示，虽然免费，但不建议使用这种方式。

公益图床

地址：https://sbimg.cn/

这是一款免费的图床平台，游客好像只能存放最多5张图片，如果需要使用最好注册一个用户，这样就可以无限存放图片了，建议使用。
